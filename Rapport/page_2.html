    Les instructions de déploiement

    Pour cloner et travailler sur ce référentiel, suivez les étapes ci-dessous :
    Assurez-vous d'avoir Git installé sur votre système. Si ce n'est pas le cas, vous pouvez le télécharger et l'installer à partir du site officiel de Git : https://git-scm.com/downloads.
    Ouvrez votre terminal ou votre interface de ligne de commande.
    Accédez au répertoire dans lequel vous souhaitez cloner le projet en utilisant la commande cd : cd chemin/vers/le/répertoire
    Clonez le référentiel en utilisant la commande git clone : git clone https://github.com/Abdoulrachard/PIL1_2223_27.git
    Accédez au répertoire du projet nouvellement cloné : cd PIL1_2223_27
    Installer Django: pip install -r requirements.py ou py -m pip install Django==4.2.2
    Demarrer le serveur d'application django: py manage.py runserver
    Ceci permettra de démarré et d'accéder à l'application en ouvrant le navigateur et en accédant l'adresse http://localhost:8000/.

    Ces étapes permettront de déployer l'application Django localement sur notre serveur Windows
